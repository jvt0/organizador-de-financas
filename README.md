# 💸 Organizador de finanças

App simples para importar extratos em CSV, consolidar transações e mostrar de forma clara gastos, recebimentos, pessoas recorrentes e insights úteis do dia a dia.

## 🛠 Stack

React, TypeScript, Vite, IndexedDB, Dexie e PapaParse.

## 📌 Escopo inicial

- importação de CSV
- suporte inicial para Inter e Nubank
- persistência local no navegador
- consolidação de transações
- visualização de pessoas recorrentes
- insights básicos
